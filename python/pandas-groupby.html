Pandas groupby

Split your data, calculate summaries, combine the results

What Is groupby?

groupby() splits a DataFrame into groups based on one or more columns, applies a function (like sum, mean, or count) to each group, and then combines the results back together. It is the pandas equivalent of SQL's GROUP BY.

Real-World Analogy

Imagine you have a pile of receipts from different stores. You sort them into piles by store name (split), add up the total for each pile (apply), and then write a summary sheet with one row per store (combine). That is exactly what groupby does.

Real Dataset Example

Suppose you have sales data for an online store:

Python import pandas as pd df = pd.DataFrame({ "date": ["2024-01-15", "2024-01-15", "2024-01-16", "2024-01-16", "2024-01-17"], "product": ["Widget", "Gadget", "Widget", "Widget", "Gadget"], "region": ["East", "East", "West", "East", "West"], "quantity": [10, 5, 8, 12, 7], "revenue": [200, 150, 160, 240, 210] })

Basic groupby: Total Revenue by Product

Python df.groupby("product")["revenue"].sum() # product # Gadget 360 # Widget 600

One line: group by product, grab the revenue column, sum it.

Multiple Aggregations

Python df.groupby("product").agg( total_revenue=("revenue", "sum"), avg_quantity=("quantity", "mean"), num_orders=("quantity", "count") ) # total_revenue avg_quantity num_orders # product # Gadget 360 6.0 2 # Widget 600 10.0 3

The .agg() method lets you compute multiple summaries at once with clear column names.

Grouping by Multiple Columns

Python df.groupby(["product", "region"])["revenue"].sum() # product region # Gadget East 150 # West 210 # Widget East 440 # West 160

This breaks down revenue by both product AND region.

Common Operations After groupby

MethodWhat It DoesExample
.sum()Total per groupTotal revenue by region
.mean()Average per groupAverage order size by product
.count()Number of rows per groupNumber of orders per customer
.min() / .max()Min or max per groupLargest sale by region
.agg()Multiple functions at onceSum, mean, and count together
.transform()Return result same size as inputAdd group mean as a new column

Pro Tip: transform vs agg

.agg() collapses groups into summary rows (like SQL GROUP BY). .transform() returns a value for every original row (like SQL PARTITION BY).

Python # Add each product's total revenue as a column (no collapsing) df["product_total"] = df.groupby("product")["revenue"].transform("sum") # date product region quantity revenue product_total # 0 2024-01-15 Widget East 10 200 600 # 1 2024-01-15 Gadget East 5 150 360 # 2 2024-01-16 Widget West 8 160 600 # 3 2024-01-16 Widget East 12 240 600 # 4 2024-01-17 Gadget West 7 210 360

Key Takeaway

Pandas groupby() follows the split-apply-combine pattern: split data into groups, apply a calculation, and combine the results. Use .agg() for summary tables and .transform() when you want to keep every row but add group-level information. It is the single most important operation for data analysis in pandas.
